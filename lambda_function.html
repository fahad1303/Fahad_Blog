üìö What is a Lambda Function?

A Lambda function, also known as an anonymous function, is a small and unnamed function defined using the lambda keyword in Python. Unlike regular functions created using the def keyword, lambda functions are written in a single line and are generally used for short, simple operations.

A lambda function can accept multiple arguments, but it can contain only one expression. The expression is evaluated and automatically returned.

üí° Key Point: Lambda functions are perfect for quick, throwaway functions!

üîß Syntax of Lambda Function

lambda arguments: expression
lambda ‚Üí Keyword used to define the function
arguments ‚Üí Input parameters (can be one or more)
expression ‚Üí A single expression whose result is returned

üìù Example 1: Lambda with Single Argument

Let's create a lambda function that squares a number:

üíª Code:

print(square(5))
‚ö° Output:
25

Here, the lambda function takes one argument x and returns its square. Simple yet powerful! üéØ

üìù Example 2: Lambda with Multiple Arguments

Lambda functions can handle multiple parameters:

üíª Code:

‚ö° Output:

In this example, the lambda function takes two arguments and returns their product. Concise and clean! ‚ú®

‚öñÔ∏è Comparing 'def' Function and Lambda Function

Using 'def' Keyword (Traditional Approach)

The traditional way of defining a function that cubes a number:

üíª Code:

    return x * x * x

‚ö° Output:

Using Lambda Function (Modern Approach)

üíª Code:

‚úÖ Observation: Both approaches produce the same result. However, the lambda version is more concise and suitable when the function logic is simple and used temporarily.

üöÄ Using Lambda Inside Another Function

Lambda functions are most powerful when used as arguments to higher-order functions (functions that take other functions as input).

Example 1: Using 'map()' Function

Apply a lambda function to each element in a list:

üíª Code:

squared_numbers = list(map(lambda x: x * x, numbers))
print(squared_numbers)
‚ö° Output:
[1, 4, 9, 16]

The map() function applies the lambda function to each element in the list. Perfect for transformations! üîÑ

Example 2: Using 'filter()' Function

Filter elements from a list based on a condition:

üíª Code:

print(even_numbers)
‚ö° Output:
[10, 20, 30]

The filter() function uses the lambda function to select only even numbers from the list. Great for data filtering! üéØ

Example 3: Using 'sorted()' with Lambda

Sort data based on a custom key using lambda:

üíª Code:

‚ö° Output:

Here, the lambda function specifies that sorting should be done based on marks (second element of each tuple). Flexible sorting made simple! üìä

‚úÖ When to Use Lambda Instead of 'def'

Lambda functions are useful in the following scenarios:

1. Short and Simple Operations

When the logic can be written in a single expression.

2. Temporary Functions

When the function is needed only once and does not require reuse.

3. Functional Programming Use Cases

When working with built-in functions like map(), filter(), reduce(), or sorted().

4. Improving Code Readability (in Small Cases)

When defining a full function using def would make the code unnecessarily lengthy.

‚ùå When NOT to Use Lambda

Avoid lambda functions in these scenarios:

‚Ä¢ Complex Logic: The logic is complex and requires multiple statements.
‚Ä¢ Documentation Needed: You need proper documentation (docstrings).
‚Ä¢ High Reusability: The function must be reused in multiple places.
‚Ä¢ Code Clarity Risk: Code readability may decrease due to overuse.

üìä Key Differences Between 'def' and Lambda

Feature `def` Function Lambda Function
Name Has a function name Anonymous (can be assigned to variable)
Number of Expressions Multiple statements allowed Only one expression
Readability Better for complex logic Better for simple logic
Reusability High Usually temporary

üéì Conclusion

Lambda functions provide a concise and efficient way to define small, single-expression functions in Python. They are especially powerful when used with higher-order functions such as map(), filter(), and sorted().

However, they should be used thoughtfully. For simple, one-line operations, lambda functions make the code cleaner and more compact. For complex logic, traditional functions defined using the def keyword are more appropriate and maintainable.
